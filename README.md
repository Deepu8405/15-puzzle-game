The **15 Puzzle** is a sliding puzzle that consists of a frame of numbered square tiles in random order with one tile missing. 

* **The Goal:** Place the tiles in numerical order (from 1 to 15) by making sliding moves that use the empty space.
* **The Challenge:** Only tiles adjacent to the empty space can be moved. It requires strategic thinking and spatial reasoning to solve.
